1. Who You Are

You are part of the Procurement team. You see every requisition in the system and you turn approved requisitions into real purchase orders.

1.1 What you can do

  • View every Purchase Requisition across all departments.
  • Add supplier quotes to a requisition.
  • Run an adjudication (three-quote comparison) to pick a winning vendor.
  • Add and maintain the vendor master list.
  • See rejected requisitions to feed back patterns to departments.
  • View reports.

1.2 What you cannot do

  • You do not approve financial requests — that is HOD / Finance / MD.
  • You cannot change the items or amounts on an approved PR.
  • You cannot manage users or system settings (that's Admin).

1.3 Your responsibilities

  1. Collect at least three quotes for requisitions above the unquoted limit.
  2. Run a fair adjudication that compares price, quality, and delivery.
  3. Document the winning vendor and the reasoning.
  4. Keep the vendor master list clean and up to date.
  5. Issue purchase orders promptly after MD approval (outside the system).

2. Quick Reference Card

One-page cheat sheet

See all approved PRs: Sidebar → Procurement → Approved Requisitions.

Add quotes: open a PR → Quotes section → Add Quote.

Run adjudication: Sidebar → Procurement → Adjudication.

Manage vendors: Sidebar → Administration → Vendors.

See rejections: Sidebar → Procurement → Rejected Requisitions.

Quote rules of thumb

  1. Below the unquoted limit: no quote needed, but adding one is always welcome.
  2. Above the unquoted limit: at least three quotes required.
  3. Sole-source (only one supplier available): attach a written justification signed by the HOD or Finance.
  4. All quotes must be in writing (PDF, email, or letterhead).

When to push back on a requisition

  • Item name too vague to source — ask the initiator to rewrite.
  • Required Date impossible given procurement lead time — ask for a realistic date.
  • Specified vendor has a bad history — raise with Finance before ordering.

3. Workflow from Your Point of View

[Initiator] --> pending_hod --> pending_finance --> pending_md --> approved ^ | | v YOU add quotes here YOU run adjudication, (often during pending_finance) issue PO outside system OPTIONAL SUB-FLOW: - Collect supplier quotes (email / PDF / website). - Upload quotes into the system via the PR detail page. - Run adjudication to pick winner. - Add winner + rationale to the PR. - After final approval, raise the PO with the vendor.

3.1 When do you step in?

The system doesn't enforce a specific stage for Procurement. In practice you should:

  • Get involved at pending_finance — this is when Finance is scrutinising the request. Add quotes and run adjudication so Finance has full data.
  • Act at approved — issue the purchase order to the winning vendor.

3.2 What the system records

  • Every quote you upload, with vendor, price, attachment.
  • The adjudication result — who you picked and why.
  • All your actions are in the audit log.

5. Task 1 — Review Approved Requisitions

5.1 Open Approved Requisitions

  1. Sidebar → Procurement → Approved Requisitions.
  2. The list shows every PR at status approved.
  3. Sort by Required Date ascending to see the most urgent first.

5.2 Open a PR detail

  1. Click a row.
  2. Check: vendor selected (if any), winning quote, adjudication outcome.
  3. If no vendor is selected yet, you need to complete that before raising a PO.
Approved requisitions list with columns ID, department, total, vendor, required date.

5.3 Reach out to the initiator (optional)

  1. If any detail is unclear, contact the initiator before placing the order.
  2. Document the clarification in the comments section of the PR.

6. Task 2 — Collect and Add Quotes

6.1 Collect quotes externally first

  1. Email, phone, or visit each candidate vendor.
  2. Ask for a written quote including:
    • Itemised unit prices
    • Currency
    • Validity period
    • Delivery lead time
    • Payment terms
  3. Save each quote as a PDF with a clear name (e.g. Quote_TechSolutions_KSB-PR-20260421.pdf).

6.2 Open the PR detail page

  1. Sidebar → Procurement → My Requisitions or Approved Requisitions.
  2. Click the PR you're working on.
  3. Scroll to the Quotes section.

6.3 Add a quote

  1. Click Add Quote.
  2. Select the Vendor from the dropdown. If the vendor is new, add them first via Administration → Vendors (see Task 4).
  3. Enter Unit Price for each item in the PR.
  4. Select Currency.
  5. Attach the Quote PDF.
  6. Enter the Validity Period (e.g. 30 days from quote date).
  7. Click Save Quote.

6.4 Repeat for each quote

  1. Add at least three quotes for comparison, if the PR requires them.
  2. If only one quote is possible (sole-source), note the reason in the Comments section.
Quotes section of a PR showing three quotes: Tech Solutions (lowest), Office Supplies Co, Furniture World.

7. Task 3 — Run Adjudication

Adjudication is the formal process of comparing quotes and choosing a winner.

7.1 Open the Adjudication page

  1. Sidebar → Procurement → Adjudication.
  2. The page lists PRs that have multiple quotes but no adjudication yet.
  3. Click the PR you want to adjudicate.

7.2 Review the side-by-side comparison

  1. The system shows each quote as a column.
  2. All non-ZMW prices are automatically converted to ZMW using the current FX rate.
  3. Look at:
    • Total cost in ZMW
    • Lead time
    • Payment terms
    • Vendor rating / tier

7.3 Pick the winner

  1. Select one quote as Winner using the radio button or "Select as winner" action.
  2. The winning quote does NOT have to be the cheapest. Cheapest, fastest, or best quality are all valid criteria — but you must justify.

7.4 Record the rationale

  1. Fill in the Rationale text box.
  2. Be specific. Examples:
    • "Tech Solutions — lowest total and shortest lead time."
    • "Office Supplies Co — not cheapest, but 30-day payment terms save cash flow."
    • "Furniture World — tier-1 vendor, 3-year warranty vs 1-year from others."
  3. Click Save Adjudication.

7.5 After adjudication

  1. The PR now has a selected vendor locked in.
  2. Finance and MD see the adjudication data during their approval.
  3. You can re-open the adjudication if needed — changes are audit-logged.
If you deviate from the cheapest option, write a clear rationale. Auditors review adjudications where the cheapest quote lost.

8. Task 4 — Manage Vendors

8.1 Open the Vendor list

  1. Sidebar → Administration → Vendors.
  2. Search for an existing vendor by name or code.

8.2 Add a new vendor

  1. Click Add Vendor.
  2. Fill in:
    • Name (required)
    • Code (short unique identifier, e.g. TSL001)
    • Category (IT Equipment, Office Supplies, Furniture, etc.)
    • Contact person and phone
    • Email
    • Tier (1 = preferred, 2 = approved, 3 = trial)
    • Rating (1-5 stars based on past performance)
    • Status — set to active
  3. Click Save.

8.3 Update a vendor

  1. Click the vendor row.
  2. Change the details.
  3. Save.

8.4 Disable (don't delete) an inactive vendor

  1. Open the vendor.
  2. Change Status to inactive.
  3. Save.
  4. Historical quotes and POs remain intact — the vendor just doesn't appear in new-quote dropdowns.

8.5 Bulk upload from Excel

  1. Download the vendor template from the Vendors page.
  2. Fill in all columns.
  3. Click Upload from Excel.
  4. The system validates and imports. Errors are reported row-by-row.
Never delete a vendor who appears on any past quote or PR — it breaks the audit trail. Disable instead.

9. Task 5 — Track Rejected Requisitions

9.1 Open Rejected Requisitions

  1. Sidebar → Procurement → Rejected Requisitions.
  2. The list shows every PR at status rejected.

9.2 Review rejection reasons

  1. Click a row.
  2. Read the approval history comments.
  3. Identify patterns: which departments have most rejections? Which items? Which reasons?

9.3 Share feedback monthly

  1. Summarise rejection patterns.
  2. Share with HODs to improve their team's initial submissions.
  3. Use the Reports page for formal numbers.

10. Task 6 — Reports

10.1 Summary Reports

  1. Sidebar → Reports & Analytics → Reports.
  2. Pick a date range (Last 30 Days for monthly view).
  3. Export to Excel or PDF.

10.2 Useful filters for Procurement

  1. By department — which departments are spending the most?
  2. By vendor — which vendors are we using most?
  3. By status — how many PRs are stuck at each stage?

10.3 FX Rates visibility

  1. Sidebar → Financial Planning → FX Rates (view-only for Procurement).
  2. Use this to check what rates are being applied to foreign-currency quotes.
  3. If rates look stale, tell Finance.

11. Troubleshooting

11.1 I can't add a quote — the Add Quote button is missing

  1. Check your role is Procurement or Admin.
  2. Check the PR is not already rejected — quotes can't be added to rejected PRs.
  3. Try refreshing the page.

11.2 A quote I uploaded doesn't appear

  1. Refresh the PR detail page.
  2. Check the file size — must be under 10 MB.
  3. Check the file format — PDF preferred; JPG / PNG / Excel / Word also accepted.

11.3 The ZMW total on adjudication is wrong

  1. FX rate may be stale. Ask Finance to refresh the rate for that currency.
  2. Then refresh the adjudication page — the new total will appear.

11.4 The vendor dropdown doesn't include the vendor I need

  1. Vendor may be set to inactive. Open Administration → Vendors, find them, and set to active.
  2. Or the vendor doesn't exist yet — add them (Task 4).

11.5 I selected the wrong winner by mistake

  1. Re-open the adjudication.
  2. Select the correct quote as winner.
  3. Save. The change is audit-logged with your name and timestamp.

12.3 Checklist before closing a PR

  • All quotes uploaded.
  • Adjudication complete with rationale.
  • Winning vendor selected.
  • PO issued externally.
  • Delivery date confirmed with vendor.
  • GRN team informed of expected delivery.
